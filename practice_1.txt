Установка и настройка Python:

Скачивание Python:
Перейдите на официальный сайт python.org и скачайте последнюю версию Python для вашей операционной системы.
Установка Python:
Запустите установочный файл и следуйте инструкциям. Убедитесь, что включили опцию "Add Python to PATH" для удобного использования Python из командной строки.
Проверка установки:
Откройте терминал или командную строку и введите:

python --version
или
python3 --version
Вы должны увидеть версию установленного Python.

Настройка окружения разработки, создание первого виртуального окружения:

В терминале введите команду:
python -m venv venv
затем:
venv/scripts/activate
